Operator Deployment LAB

The following video will provide the step-by-step guide for installing the Ansible Automation Platform 2.5 operator:

  1. Log in to the Red Hat OpenShift Container Platform.

  2. Navigate to Operators OperatorHub.

  3. Search for Ansible Automation Platform and click Install.

  4. Select a stable-2.5.

  5. Select namespace as default which will be aap.

  6. Click Install.
